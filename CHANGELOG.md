# Changelog - QGIS Layout Item Selector

バージョン表記: VA.B.C
- 例: V2.0.0（A=2, B=0, C=0）

## バージョン番号の意味
- **A**: QGIS本体のバージョンアップに伴う本体の修正
- **B**: UIの変更（プラグインの追加等含む）、本体の機能追加
- **C**: プロファイル・プラグイン、本体の修正
